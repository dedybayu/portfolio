Source Code Open Website

Project Overview

PresApp is a Laravel-based system designed for recording student achievements and providing intelligent recommendations for upcoming competitions. The platform integrates Decision Support System (DSS) methodologies including TOPSIS, SAW, and PSI to calculate and suggest the most suitable participants. This system has been officially registered for Copyright.
