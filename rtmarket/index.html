RT Market Connect

Project Overview

RT Market Connect is a community management system and local marketplace built with a decoupled architecture. It utilizes Golang as a high-performance API gateway, Flutter for a responsive mobile interface, and a Computer Vision (Python) module for automated fruit classification within the marketplace engine.
