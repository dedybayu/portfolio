Project Overview

NetMonitor is a real-time infrastructure monitoring dashboard integrating SNMP Polling and Proxmox API. It provides a comprehensive solution for network administrators to visualize network topology, monitor VM/Node status, and track centralized device performance. Features include dynamic SNMP discovery, interactive topology maps, and seamless integration with virtualization environments.
